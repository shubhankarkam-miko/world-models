Diffuser: planning as denoising

Janner, Du, Tenenbaum & Levine — Planning with Diffusion for Flexible Behavior Synthesis

Instead of learning a dynamics model and handing it to a separate trajectory optimizer, this paper trains a diffusion model directly on whole trajectories. Sampling from the model is planning — rewards steer the sample via classifier-style guidance, and goals/constraints are enforced via inpainting. Inside the Diffuser U-Net

Diffuser's denoising network is a U-Net — the same architecture family used in image diffusion models — but with 2D spatial convolutions replaced by 1D temporal convolutions that slide along the planning horizon. Six residual blocks like the one on the right make up the network (Figure A1 in the paper).

t

A diffusion-timestep embedding t passes through a fully-connected layer and is added to the activations right after the first temporal convolution. A skip connection carries the block's input straight to the output, added back in after the second group-norm + Mish.
